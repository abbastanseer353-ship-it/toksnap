# **App Name**: TokSnap

## Core Features:

- Social Discovery Feed: A scrollable masonry grid showing the latest image uploads from the community.
- Quick Upload Workflow: Streamlined interface to upload photos directly from your device with progress tracking.
- Cloud Metadata Store: Use Firestore to manage image metadata, timestamps, and user associations.
- Smart Vision Tool: A generative AI vision tool that analyzes uploaded photos to automatically suggest relevant alt-text and categorization tags.
- Material Profile: A personalized section to view and manage your uploaded content, adhering to Material Design 3 guidelines.
- Snap Persistence: Scalable cloud-based storage system for hosting high-resolution image assets.
- Micro-Reaction System: Haptic-feedback-driven interaction layer allowing users to quickly 'snap-react' to content in the feed.

## Style Guidelines:

- Primary Color: Vivid Orchid (#C166F2), evoking the energy and creativity of a fast-paced social platform.
- Background Color: Deep Charcoal Plum (#17141A), providing a sleek, dark canvas that makes imagery pop.
- Accent Color: Periwinkle Mist (#9EA2FF), used for navigation highlights and interactive components.
- Headline Font: 'Poppins' (Sans-serif) for its geometric precision and modern punch. Body Font: 'Inter' (Sans-serif) for high readability in dense image feeds.
- Material Design system icons using 'rounded' variants to ensure a soft, accessible, and friendly interface.
- Bottom-heavy navigation focus typical of mobile-first Android apps, using a persistent Navigation Bar for primary views.
- Staggered 'Slide-In' transitions for feed items and 'Elastic-Scale' micro-interactions for snapping reactions.
